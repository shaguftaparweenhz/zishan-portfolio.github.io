# Zishan's Portfolio Website

A modern, responsive portfolio website showcasing Zishan's skills, projects, and contact information.

## Features

- Fully responsive design that works on all devices
- Modern UI with smooth animations
- Interactive navigation with smooth scrolling
- Project showcase section
- Contact form with validation
- Mobile-friendly navigation menu

## Technologies Used here

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6)
- Font Awesome for icons
- Google Fonts (Poppins)

## How to Customize

1. **Update Personal Information:**
   - Edit `index.html` to update your name, title, and bio
   - Replace placeholder images with your actual photos
   - Update contact information in the contact section

2. **Add Your Projects:**
   - Modify the "Projects" section in `index.html` with your actual projects
   - Update project titles, descriptions, and tech stack

3. **Customize Colors:**
   - Edit `styles.css` to change the color scheme
   - Primary color is defined as `#4a6cf7` - replace with your preferred color

4. **Connect Contact Form:**
   - The current form shows an alert - connect to a backend service to receive messages

## File Structure

```
zishan-portfolio/
│
├── index.html          # Main HTML file
├── styles.css          # Stylesheet
├── script.js           # JavaScript functionality
└── README.md           # Documentation
```

## How to Run

1. Open `index.html` in any modern web browser
2. Or host it on any static hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Deployment

To deploy this portfolio:

1. Push the code to a GitHub repository
2. Use GitHub Pages, Netlify, or Vercel to host the site
3. Update DNS records if using a custom domain
